Running The Project:

Front End : Java
IDE       : Eclipse

The folder PoemFeatureExtractor contains the source code of the project. The project was done using Eclipse IDE. The different files of the projects
will be present inside the folder in subdirectories.
To run the project, go to the Executable folder and double click poem.jar. A window opens. Give the input poem in first text area and click Show features.
The output wil be displayed in the second text area.

----------------------------------------------------------------------------------------------------------------------------																						
To Run Project in command line

java - jar Executable/poem.jar
